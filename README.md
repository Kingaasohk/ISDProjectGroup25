# A & A's Taxi Booking System  

This is the source code for A & A's Taxi Booking System, the group project made for Assignment 2 for the Introduction to Software Development at CTS College/University of Bedfordshire by Group 25 (Aaron Rambeharrysingh and Apiffany Mitchell).

## Virtual Environment Setup 
Our project can be executed without a virtual environment (not required), but recommended to run it within a virtual environment to recreate the environment we had when developing and testing our code.
1. Create Virtual Environment:
   
 On Windows:
 python -m venv venv

 On macOS/Linux:
 python3 -m venv venv

## Build Instructions

2. Activate the Virtual Environment

On Windows (in Command Prompt or PowerShell):
.\venv\Scripts\activate

On macOS/Linux: 
source venv/bin/activate
   

Clone the Repository into a blank folder with this command:
`https://github.com/Kingaasohk/ISDProjectGroup25.git`
<br>
**OR**
<br>
Paste the contents of the provided ZIP file into a blank folder.
